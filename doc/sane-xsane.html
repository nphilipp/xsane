What is XSane?

XSane is a new improved frontend for SANE.
 

     

    
 
 

Whats new in XSane 0.20:

  • Improved Preferences/setup
  • Added selection of up to 10 printers in photocopy mode.
  • Added gamma correction for printer.
  • Improved gamma correction for preview (monitor).
  • If file could not saved an error message is printed.
  • For more news read CHANGES.XSANE.

    • Features:

  • Scan and photocopy function.
  • Works as GIMP-plugin.
  • Unique user interface for all scanners.
  • 8 bit output formats: pnm, png, postscript, jpeg and tiff.
  • Experimental 16 bit support.
  • 16 bit output formats: pnm(ascii), png and raw(comparable to pnm raw).
  • Automatic increase of filename counter (use filename of the form: image-001.ext)
  • Changes of color enhancement  (gamma, brightness, contrast, white-, gray- and black point) are updated in preview window without a new preview-scan!
  • Gamma correction for preview (monitor) and printer.
  • Color enhancement for scanners/backends that do not support custom gamma.
  • Autoadjust-function for gamma, contrast and brightness.

    • Requirements:

  • sane-1.0 installed
  • gtk-1.0.x installed (1.1.x and 1.2.x do not work)
  • for jpeg-support: jpeglib
  • for png-support: libpng and zlib
  • for tiff-support: libtiff

    •  

    How to compile/install:

  • Unpack archiv.
  • Enter xsane-0.xx directory.
  • To compile call ./configure and make.
  • To install call make install as root.
  • Call xsane.
  • If you get an error message  that libsane-dll.so.1 is not found, you have to tell your system where the SANE-libraries are installed. For linux: edit /etc/ld.so.conf and add a line with the path to the libraries (normally /usr/local/lib/sane), then call ldconfig as root.
  • For online help you need netscape, on some SuSE systems there only is "Netscape" and no "netscape" in the path, in this case you have to set a symbolic link:
    • ln -s /usr/bin/Netscape /usr/bin/netscape

Was ist XSane?

XSane ist ein graphisches Scanner Frontend für SANE (Scanner Access Now Easy). Es benutzt die SANE-Bibliothek, um Scanner anzusteuern. Das bedeutet, dass XSane selbst keine Scanner unterstützt. Hier können Sie herausfinden, welche Scanner von der SANE-Bibliothek unterstützt werden.

XSane wurde entwickelt, um Bilder mit Scannern einzulesen (es gibt andere Geräte wie Kameras und Videogeräte, die von SANE unterstützt werden, aber XSane ist für diese Geräte nicht konzipiert.) Sie können in eine Datei scannen, photokopieren, Faxe erstellen und XSane aus GIMP als gimp Plugin starten.

Einstellungen:

Bevor Sie mit dem Scannen beginnen, sehen Sie sich bitte alle Einstellungsseiten an. XSane könnte nicht korrekt funktionieren oder Sie können nicht alle Funktionen nutzen, wenn Sie XSane nicht richtig konfigurieren.

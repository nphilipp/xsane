Index of xsane documentation

 
  • XSane is a GUI (graphical user interface) for SANE (Scanner Access Now Easy). It is designed for acquiring images with scanners (there are other devices like cameras supported by SANE, but XSane is not designed for that purpose). XSane can be run in four different modes:
  • Scan mode
  • Copy mode
  • Fax mode
  • Gimp plugin
  • There are several options that are scanner or backend dependant. XSane does not know much about these options, it only creates Gadgets to give the user the possibilty to take advance of these options. You find these options in the:
  • Standard options window (for general options)
  • Advanced options window (for options that should be used by advanced users)
  • Before you do a scan you normally do a preview scan to do color correction and select the desired part of the scan surface. There are  several functions in the Preview-window:
  • Pipette white
  • Pipette gray
  • Pipette black
  • Unzoom
  • Zoom out
  • Zoom in
  • Undo zoom
  • Select visible area
  • Select scanarea
  • Move scanarea
